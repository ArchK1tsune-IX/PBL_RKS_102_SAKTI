Welcome to SAKTI

Sistem Aplikasi Kriptografi dan Steganografi Interaktif

Welcome To PBL RKS-102!

SAKTI is a Project Based Learning (PBL) developed by PBL-RKS102 Team. SAKTI is created with Python Flask, Bootstrap HTML, and MySQL Databases.

It is a system that can encrypt/decrypt text and files using classic ciphers. Besides that, you can hide and extract data from image files using modern ciphers.
